Implentação CSS Cascading Style Sheets (abreviado CSS) é um mecanismo para adicionar estilos (cores, fontes, espaçamento, etc.)

A uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags.

Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos.
